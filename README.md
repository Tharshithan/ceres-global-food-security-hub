**Ceres** is a cutting-edge web platform dedicated to monitoring, analyzing, and visualizing global food security metrics. Named after the goddess of agriculture, this hub leverages Artificial Intelligence and real-time data to provide actionable insights into resource distribution and agricultural sustainability.

## ✨ Key Features

* **🤖 AI-Powered Analysis:** Integrated with the **Google Gemini API** to provide smart summaries of complex food security datasets.
* **📊 Interactive Data Visualization:** Dynamic charts and graphs powered by **Recharts** to track global food trends.
* **🔥 Real-time Database:** Backed by **Firebase Firestore** for instant data synchronization across users.
* **✨ Modern UI/UX:** A smooth, responsive interface built with **Tailwind CSS** and animated using **Framer Motion**.
* **📱 Fully Responsive:** Designed to provide a seamless experience across desktop, tablet, and mobile devices.

---

## 🛠️ Tech Stack

### Frontend
* **React 19:** For building a component-based, high-performance UI.
* **TypeScript:** Ensuring type safety and scalable code.
* **Vite:** A lightning-fast build tool for modern web development.

### Backend & Services
* **Firebase:** Handles data storage (Firestore) and project configuration.
* **Google Generative AI:** Powers the hub's analytical capabilities via the Gemini model.

### Visualization & Styling
* **Lucide React:** Beautiful, consistent iconography.
* **Recharts:** For complex, responsive data visualisation.
* **Tailwind CSS:** For a clean, modern utility-first design.
